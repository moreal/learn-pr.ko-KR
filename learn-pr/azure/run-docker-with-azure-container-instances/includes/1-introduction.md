컨테이너는 패키지, 배포 및 클라우드 응용 프로그램 관리는 훌륭한 방법입니다. Azure Container Instances를 통해 virtual machines를 관리 하지 않고 더 높은 수준의 서비스를 Azure에서 컨테이너를 실행할 수 있습니다.
