이제 컨테이너를 사용하는 방법에 대한 기본 사항을 알아보았습니다. 로컬 Windows, MacOS 또는 Linux 컴퓨터에서 이미지를 만들고 실행할 수 있는 Docker를 설치했습니다. 그런 다음, 몇 가지 기본 Docker 명령에 대해 알아보았습니다. 마지막으로, 사용자 고유의 이미지를 만들고 Docker 허브에 이 이미지를 저장하는 방법을 알아보았습니다.
