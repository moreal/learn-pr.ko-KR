글로벌 제약 회사에서 네트워크 관리자로 일한다고 가정해 보겠습니다. 이 회사는 독점적인 상업 기밀 의약품에 대한 수식을 만들기 위해 유전자 배열을 적용합니다. 이러한 수식은 귀사의 전 세계 제조 공장에서 사용됩니다. 귀사는 민첩성을 높이기 위해 Microsoft Azure를 구현하고 유전자 배열 함수의 일부를 클라우드에서 실행되는 가상 머신으로 전송하려고 합니다. 데이터 처리의 결과는 전 세계 여러 지역에서 사용할 수 있어야 합니다. 이러한 처리는 현재 각 지역의 데이터 센터를 연결하는 데 사용되는 VPN(가상 사설망)을 사용하여 온-프레미스 데이터 센터에서 수행됩니다.

Microsoft Azure가 네트워킹을 구현하는 방법을 평가하고 온 프레미스 데이터 센터와 Microsoft Azure 사이 및 Microsoft Azure 지역 간의 데이터 전송에 적절한 보안을 제공하는지의 여부를 확인하는 임무가 귀하에게 주어졌습니다. Azure Virtual Network, Azure VPN Gateway 및 Azure ExpressRoute 기술을 사용할 수 있습니다.
