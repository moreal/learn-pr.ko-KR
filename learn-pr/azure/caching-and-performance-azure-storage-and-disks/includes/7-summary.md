이 모듈에서는 Azure 디스크 캐싱과 잠재적으로 성능이 향상되는 방법을 알아보았습니다. Azure Portal 및 Azure PowerShell을 사용하여 VM에 대한 디스크 캐싱을 관리했습니다.

Azure VM 디스크 캐싱 전략을 마련하고 나면 스크립트 및 템플릿을 사용하여 최적의 디스크 캐시 설정으로 새 VM 및 디스크를 빠르고 쉽게 배포할 수 있습니다.
